### Ships

- **`regions: Regions`** — region aliases + canonical region names per
  cloud provider (`aws`, `azure`, `gcp`, `cloudflare`). Single
  source-of-truth for region resolution across the `@adaptiveworx/iac-*`
  packages.
- **`config/regions.json`** — the underlying data file, also exposed via
  the `./regions` subpath export.
- **`generated/schemas/`** — Zod-derived JSON Schemas + OpenAPI envelope
  (regenerated by `iac-core/schemas/generators/generate-schemas.ts`).

### No runtime dependencies

The package is pure data + types. Zod is only required if you use the
generated JSON Schemas with a Zod runtime validator.
